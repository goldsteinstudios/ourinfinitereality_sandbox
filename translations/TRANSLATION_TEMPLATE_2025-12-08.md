# Chapter Translation Template

**Purpose:** Documentation, not persuasion. Show pattern, don't argue.

## Structure

### 1. Header
```markdown
# Chapter [N]: [Title]

**Guodian:** [Bundle:Slip] (~300 BCE) [if applicable]
**RSM Function:** [One-line description]
```

### 2. Text Block
```markdown
## Chinese Text
[Full Chinese text]

## Pinyin
[Full pinyin with tone marks]

## Structural Translation
[Direct, clean translation showing operations]
```

### 3. Radical Analysis
```markdown
## Key Operations

| Character | Radicals | Operation |
|-----------|----------|-----------|
| [Char] | [Components] | [What it does] |

[Brief explanation of geometry, no arguments]
```

### 4. Cross-References
```markdown
## Pattern Connections

| Chapter | Connection | Operation |
|---------|-----------|-----------|
| [N] | [Brief description] | [Shared pattern] |
```

### 5. Formula/Summary
```markdown
## The Operation

[Clean statement of what this chapter documents]
[No defense, no refutation, just show]
```

### 6. Appendix References
```markdown
---

**See Appendix:**
- [Letter]: [Topic] - [Brief description]

**Guodian Notes:**
[If applicable, note variants and significance]
```

## Example: Chapter 40

### ✅ Good (Clean Documentation)

```markdown
# Chapter 40: The Oscillation Engine

**Guodian:** Bundle A, Slip A37 (~300 BCE)
**RSM Function:** Core engine - reversal and yielding

---

## Chinese Text

反者道之動
弱者道之用

## Pinyin

fǎn zhě dào zhī dòng
ruò zhě dào zhī yòng

## Structural Translation

Reversal is pattern's movement.
Yielding is pattern's function.

---

## Key Operations

**反 (fǎn) - Reversal**
- 又 (hand) + 厂 (cliff) = hand at cliff edge, turning back
- O→G→P→O recursion
- The return operation

**弱 (ruò) - Yielding**
- 弓 (bow) + 弓 (bow) = two bows, flexible tension
- Compression-release cycle
- Functional capacity through flexibility

---

## Pattern Connections

| Chapter | Connection |
|---------|-----------|
| 16 | 各復其根 - Each returns to root |
| 25 | 大→逝→遠→反 - Sphere proof |
| 22 | 曲則全 - Curve completes |

---

## The Formula

```
反 (reversal) = pattern's movement (動)
弱 (yielding) = pattern's function (用)

Movement through return.
Function through flexibility.
```

---

**Guodian:** Confirmed c. 300 BCE, most concise statement of core principle
**See Appendix:** B - Archaeological validation of oscillation model
```

## Length Guidelines

**Main Text:**
- Chapter header: 1-2 lines
- Text block: As needed
- Radical analysis: 5-20 lines
- Cross-references: 3-10 entries
- Formula/Summary: 3-10 lines
- **Total: 50-200 lines**

**Appendices:**
- Per topic: 50-500 lines
- Separated from main text
- Referenced but not blocking
- Available for depth, not required for understanding

## Examples of Clean Documentation

### Water
```markdown
水善利萬物而不爭

Water optimally cuts-paths-through all things without fighting over positions.

利 = 禾 (grain) + 刂 (blade) = scythe operation (π-arc through substrate)
爭 = 爪 + 爪 + 彐 = hands fighting over position

Water demonstrates non-occupation.
```

### Governance
```markdown
為無為則無不治

Engage the act-rest-act rhythm, then everything self-organizes.

為無為 = Act-void-act (oscillation pattern)
治 = 氵(water) + 台 (platform) = build structures supporting flow

Pulse rhythm enables self-organization.
```

### Self-Transformation
```markdown
萬物將自化

All things will self-transform.

自 = self (origin marker, agent is internal)
化 = 亻+ 匕 + 匕 = person transforming states

Transformation from within, not imposed from without.
```
