# Glossary

*Key terms with structural definitions*

---

## Core Terms

terms:道
:   **dào** — Pattern, way. 辶 (walking) + 首 (head) = walking with head leading. The continuous aligned movement. NOT a mystical force—the invariant structure that persists across frames.

terms:德
:   **dé** — Alignment, power. 彳 (step) + 直 (straight) + 心 (heart) = heart stepping straight. Operational alignment with 道, NOT "virtue" as moral quality.

terms:無為
:   **wú wéi** — Non-forcing action. Scythe-action vs knife-action. Arc rotation with the field, not linear force against it. The efficient harvester's swing.

terms:自然
:   **zì rán** — Self-so, spontaneous. What emerges from a system's own nature without external imposition. NOT "nature" as environment.

terms:常
:   **cháng** — Constant, invariant. What remains the same across frames. Guodian uses 恆 (héng). Frame-independent truth.

terms:玄
:   **xuán** — Paradoxical origin. 亠 (top) + 幺 (thread) = subtle thread covered. Where axes cross, where distinction hasn't yet operated. The generative center.

---

## Operators

operators:利
:   **lì** — Scythe harvest, path-cutting benefit. 禾 (grain) + 刂 (blade) = blade arcing through field. The π-operation executed physically.

operators:反
:   **fǎn** — Reversal, arriving at opposite. 厂 (cliff) + 又 (hand) = hand reaching back. The return arc completing 2π. NOT "going backward"—the cycle completing.

operators:治
:   **zhì** — Governance as hydraulics. 氵 (water) + 台 (platform/aqueduct) = water channeled through infrastructure. Build correctly, let gravity work.

operators:和
:   **hé** — Harmony, distributed flow. 禾 (grain) + 口 (mouth) = grain through mouths. Proper distribution, not forced agreement.

operators:知
:   **zhī** — Knowing, targeted articulation. 矢 (arrow) + 口 (mouth) = arrow given mouth. Directional, intentional attention.

operators:名
:   **míng** — Name, articulation. 夕 (evening) + 口 (mouth) = mouth articulating darkness. Making explicit what was implicit.

---

## Substrates

substrates:禾
:   **hé** — Grain radical. Pictograph of drooping grain head on stalk. The organic field that receives the blade operation.

substrates:氵
:   **shuǐ** — Water radical. Central current + side eddies. The pattern of natural flow. Water IS 無為 in physical form.

substrates:心
:   **xīn** — Heart radical. The interior state that aligns (德) or misaligns with pattern.

substrates:刂
:   **dāo** — Blade radical. Curved stroke + vertical hook = curved blade + two-handed handle. Complete scythe geometry. NOT a knife (linear push).

---

## Structural Positions

terms:無
:   **wú** — Void, absence, potential. That which cannot be grasped. The generative emptiness that provides function (用).

terms:有
:   **yǒu** — Presence, form, manifest. 又 (hand) + 月 (flesh) = hand holding flesh. That which CAN be grasped. Provides constraint (利).

terms:可
:   **kě** — Expressible, frame-dependent. 丁 (nail) + 口 (mouth) = fixed point given mouth. What can be articulated from a particular position.

terms:妙
:   **miào** — Relational patterns, flows. 女 (woman) + 少 (few) = feminine + subtle. What flows INTO and THROUGH. Observation stance toward connections.

terms:徼
:   **jiào** — Boundaries, edges. 彳 (step) + 敫 (boundary-marker). Where things STOP. Observation stance toward form-edges.

terms:剛
:   **gāng** — Rigid, hard. 岡 (ridge) + 刂 (blade) = blade meeting ridge. What resists the arc. Property from fast quench.

terms:柔
:   **róu** — Flexible, soft. 矛 (spear) + 木 (wood) = workable wood. What CAN engage the arc. Property from slow cooling.
